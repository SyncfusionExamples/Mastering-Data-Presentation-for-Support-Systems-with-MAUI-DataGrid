# Mastering-Data-Presentation-for-Support-Systems-with-MAUI-DataGrid

In a .NET MAUI app, use the `Google.Cloud.Firestore` SDK to connect to Firebase Firestore using a credentials JSON file. Query the Firestore collection, convert each document to a model (SupportTicket), and return an ObservableCollection. Bind this collection to the SfDataGrid to display the data in a structured grid format.
